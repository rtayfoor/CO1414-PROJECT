# CO1414-PROJECT
Overview:
This project contains my version of the website for selling products. 
Table of Contents:
- Features
- Bugs/issues
- Extra notes

Features:
- header and footer in all pages with contact details and primary pages.
- home page (index.html) with 2 videos, a paragraph and my title. 
- a product page (products.html) with a list of all products available in website with details (item.html), along you can also see which are out of stock and which are in stock with the important function of not letting you add an item in the cart if out of stock.
- i included my cart (cart.html) which displays the user's selected items. it allows the user to remove items from the cart or in a swift way remove all items when a lot are added.
